# 🧠 Signify Connect AI – Sign Language Translator

An AI-powered system that converts sign language gestures into text and voice in real-time.

## 🚀 Features
- Real-time gesture recognition using OpenCV
- CNN-based model for accurate prediction
- Converts gestures to text + speech
- Supports Indian Sign Language (ISL)

## 🛠️ Tech Stack
- Python
- OpenCV
- Machine Learning (CNN)
- Text-to-Speech (TTS)

## 📊 Impact
- Helps bridge communication gap for hearing/speech impaired individuals
- Can be used in hospitals, schools, and public services

## 🔥 Future Scope
- Multi-language support
- Mobile app integration
- Cloud deployment
